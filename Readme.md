## Local setup

### Step 1 - Clone the repo

```bash
git clone https://github.com/HLrahul/Taskflo.git
```

```bash
cd taskflo
```

### Step 2 - Install deps for server and run

```bash
cd server
```

```bash
npm i
```

### Step 3 - Install deps for client and run

```bash
cd client
```

```bash
npm i
```
